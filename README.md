# Django RESTFUL API

Backend API using RESTFUL Django framework and Test Driven Development

## Run project

  ```bash
  docker-compose build
  ```

  ```bash
  docker-compose up
  ```
  
## Run tests and flake8 validations 

  ```bash
  docker-compose run app sh -c "python manage.py test && flake8"
  ```
